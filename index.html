JavaScript Básico

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

Variáveis


// Declaração de variáveis
let nome = "João";
const idade = 30;
var profissao = "Desenvolvedor";

console.log(nome); // João
console.log(idade); // 30

Estruturas de Controle


// Condicional if-else
let temperatura = 25;

if (temperatura > 30) {
    console.log("Está muito quente!");
} else if (temperatura > 20) {
    console.log("Temperatura agradável");
} else {
    console.log("Está frio");
}

// Loop for
for (let i = 0; i < 5; i++) {
    console.log("Iteração: " + i);
}
Dica: Use let e const em vez de var para declarar variáveis, pois eles têm escopo de bloco.

Python Básico

Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte.

Variáveis e Tipos


# Declaração de variáveis
nome = "Maria"
idade = 25
altura = 1.68
is_programmer = True

print(nome)  # Maria
print(type(idade))  # <class 'int'>

Estruturas de Controle


# Condicional if-elif-else
temperatura = 22

if temperatura > 30:
    print("Está muito quente!")
elif temperatura > 20:
    print("Temperatura agradável")
else:
    print("Está frio")

# Loop for
for i in range(5):
    print(f"Iteração: {i}")
Dica: Python usa indentação para definir blocos de código, ao invés de chaves como em outras linguagens.

C++ Básico

C++ é uma linguagem de programação compilada multi-paradigma (seu suporte inclui linguagem imperativa, orientada a objetos e genérica) e de uso geral.

Variáveis e Tipos


// Declaração de variáveis
#include <iostream>
using namespace std;

int main() {
    string nome = "Carlos";
    int idade = 35;
    double altura = 1.75;
    bool is_programmer = true;
    
    cout << nome << endl;  // Carlos
    return 0;
}

Estruturas de Controle


// Condicional if-else
int temperatura = 18;

if (temperatura > 30) {
    cout << "Está muito quente!" << endl;
} else if (temperatura > 20) {
    cout << "Temperatura agradável" << endl;
} else {
    cout << "Está frio" << endl;
}

// Loop for
for (int i = 0; i < 5; i++) {
    cout << "Iteração: " << i << endl;
}
Dica: C++ é uma linguagem fortemente tipada e compilada, o que significa que os erros são detectados em tempo de compilação.
